 Script will help to find some string in an files.

 Installation:

 1. Set an nick and password in file find-string.php in the lines:
	$nick = "admin"; // Attention!!! Change it!
	$password = "1234"; // Attention!!! Change it!
 2. Copy file find-string.php to root folder of your website.
 3. Open file http://your web site/find-string.php in browser.
