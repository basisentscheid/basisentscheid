 Version 0.1.0

 Описание: Скрипт предназначен для поиска строк в коде, с возможностью использования регулярных выражений.

 Установка:

 1. Укажите имя и пароль для доступа к скрипту в файле find-string.php в строках:
	$nick = "admin"; // Attention!!! Change it!
	$password = "1234"; // Attention!!! Change it!
 2. Скопируйте файл find-string.php в корень вашего сайта.
 3. Откройте файл http://your web site/find-string.php в броузере.
